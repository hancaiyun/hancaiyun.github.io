# 个人网站 Demo

这是为 `hancaiyun/hancaiyun.github.io` 提供的个人网站示例。把这些文件放在仓库根目录（`index.html`、`styles.css`、`script.js`、`README.md`），然后推送到 GitHub。

如何部署
1. 将文件添加到仓库根目录。
2. 推送到默认分支（通常是 `main`）。
3. 在仓库 Settings -> Pages 中确认 Source（分支）为你刚推送的分支和 `/ (root)`。
4. 等待 GitHub Pages 构建并发布（通常几分钟内生效）。

自定义建议
- 替换 index.html 中的文本（名字、简介、邮箱、社交链接）。
- 在 `script.js` 中替换 `projects` 和 `posts` 数组为你的实际内容，或接入静态生成流程。
- 想使用博客功能可改用 Jekyll/Hugo，或用 GitHub Actions 自动把 Markdown 转为 HTML。
